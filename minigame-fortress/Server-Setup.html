# Server Setup

이 문서에서는 Fortress (opens new window)의 서버 설정을 다룹니다.

# 서버 구동 스크립트 사용

Releases (opens new window)에 있는 server.ps1 (opens new window)을 실행하는 방법입니다.

Linux 환경의 경우 PowerShell을 설치하셔야 작동됩니다.

# 스크립트 실행

새 디렉터리에서 실행하는 것을 권장합니다.

서버 실행을 원하시는 디렉터리에서 .\server.ps1 명령을 사용하여 서버를 구동하세요. 자동으로 서버가 구축이 될 겁니다.

월드 파일은 [Releases]에 있는 [world.zip]에서 구하실 수 있습니다.

빌드에 문제가 생기거나 서버 실행에 문제가 생긴 경우 Server Error FaQ나 Minecraft Gradle FaQ를 참고하여 주시길 바랍니다.

  • 실행영상

    • 환경 : Windows 10 1809
